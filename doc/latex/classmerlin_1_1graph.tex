\subsection{Detailed Description}
The graph structure used by \hyperlink{classMerlin}{Merlin}. 

The graph contains N {\itshape nodes} (indexed 0..N-\/1) and E {\itshape edges}, with bidirectional indexing (0..2\+E-\/1).
