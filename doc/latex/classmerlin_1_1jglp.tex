\subsection{Detailed Description}
Join-\/\+Graph Linear Programming (ie, Weighted mini-\/buckets for M\+A\+P inference)

Tasks supported\+: M\+A\+P

J\+G\+L\+P is a specialization of the weighted mini-\/bucket algorithm to the M\+A\+P inference task. It builds a join graph by running the mini-\/bucket algorithm schemetically, and then it connects the mini-\/buckets residing in the same bucket. J\+G\+L\+P shifts costs along the edges of the join graph by matching the max marginals of any two clusters connected by an edge. In practice, the algorithm typically tightens the M\+A\+P upper bound, however this is not a guarantee. It keeps track of the tightest upper bound found and its corresponding assignment.
